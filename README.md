# Python_Assignment

Powerlifting weak spot fix

My project is a tool that assists people in gaining strength in the gym. In particular, it is focused towards the three "lifts" that make up the sport of powerlifting: The Squat; The Bench Press; and the Deadlift. 

As is explained in my about page, an understanding of physics is crucial to lifting heavy weights in the gym. My about page briefly monologues on the purpose of my applicaiton, then my other pages take information from the user, which figures out what their weaknesses are in relation to their lifts.

From there, the user is directed to the concepts page which explains the different types
of techniques and tools which are used to cheat the physics. Then the user is directed to the quiz, where my application figures out what they need to work on, then gives the user exercises which they should focus on to improve their weaknesses.
